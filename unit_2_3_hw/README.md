# Модуль 2.3. Домашнє завдання: Superloop та незалежні задачі

## Домашнє завдання: Superloop та незалежні задачі

- Код і результат виконання на GitHub.
- Посилання на репозиторій в LMS.

## Завдання

Керувати кількома незалежними процесами в одному loop() без блокування.

Є 3 світлодіоди:

- LED1 блимає кожні 200 мс
- LED2 блимає кожні 500 мс
- LED3 блимає кожні 1000 мс

## Вимоги

- Заборонено використовувати delay()
- Заборонено використовувати while() у loop()
- Дозволено використовувати millis()
